Cross Cutting layer

Handles concerns that permeate multiple layers of the application (cross-cutting concerns).
These concerns are not directly related to the core business logic but are essential for the overall functionality and maintainability of the system.

* Logging
* Security (authentication, authorization, role-based access control...)
* Caching
* Error handling (exceptions and log errors)
* Validation  (enforces data integrity and consistency through validation rules)
* Auditing (tracks changes to data and system activities for compliance and security purposes)
* Performance optimization (techniques like profiling and caching to optimize application performance)
